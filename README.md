# Backup Automatizado de Equipos de Red

Script en Python para realizar backups de switches HP mediante Netmiko.

## Características

- Conexión por Telnet
- Uso de variables de entorno (.env)
- Backup automático
- Integración con OneDrive (SharePoint)

## Instalación

pip install -r requirements.txt

## Configuración

Crear archivo .env:

SWITCH_PASSWORD=tu_password

## Ejecución

python main.py
